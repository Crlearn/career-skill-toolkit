# BOSS 打招呼话术库

> 国内招聘平台（BOSS直聘、拉勾、猎聘）的沟通以聊天为主，打招呼话术直接决定 HR 是否回复。

---

## 一、话术阶段

| 阶段 | 时机 | 目标 |
|------|------|------|
| **首次打招呼** | 投递简历时 | 3 秒内让 HR 想点开你的简历 |
| **已读不回跟进** | 对方已读但 3 天未回复 | 换角度重新引起注意 |
| **面试后感谢** | 面试结束后 2 小时内 | 加深印象+补充亮点 |
| **Offer 谈薪** | 收到 Offer 但薪资不满意 | 争取更好条件 |

---

## 二、首次打招呼模板

### 模板 1：硬技能直击型（最推荐）

```
您好，看到贵司[岗位名]非常匹配：
1、[硬技能1：含技术栈+具体经验]；
2、[硬技能2：含工具/方法+成果]；
3、[经验描述：具体场景+交付物]。
期待与您进一步沟通！
```

**示例**（Java 开发工程师）：
```
您好，看到贵司目标岗位A岗位非常匹配：
1、熟练掌握 XX框架 微服务开发，具备 XX数据库 设计与 XX缓存 实践经验；
2、熟悉 XX工具A、XX工具C 等 AI 编程工具，能利用 AI 辅助编码与需求分析；
3、独立完成 XX管理系统 从需求分析到代码落地全流程，包含 3 个微服务模块交付。
期待与您进一步沟通！
```

### 模板 2：项目亮点型

```
您好！我对贵司[岗位名]很感兴趣。
我独立完成过[项目名]，使用[技术栈]，实现了[核心功能]。
持有[证书名]证书，[学校名][专业]在读，预计[时间]毕业。
方便聊聊吗？
```

### 模板 3：简洁型（HR 看消息列表时只看前两行）

```
您好，[学校][专业]应届生，熟练掌握[核心技能]，有[项目类型]项目经验，期待交流！
```

## 三、已读不回跟进模板

**原则**：不追问"为什么不回复"，而是提供新价值。

```
[HR姓名]您好，之前投递了[岗位名]，补充一下：
我最近在[做的新事情/学的新技术]，和贵司[业务方向]很相关。
如果岗位还在招，期待聊聊！
```

**示例**：
```
李女士您好，之前投递了目标岗位A，补充一下：
我最近在用 Docker 部署微服务项目，做了完整的 CI/CD 流程，和贵司的技术栈很相关。
如果岗位还在招，期待聊聊！
```

## 四、面试后感谢模板

```
[面试官姓名]您好，感谢今天的面试交流！
关于[面试中讨论的某个技术点]，我补充一下：[简短补充，1-2句话]。
期待后续消息，祝工作顺利！
```

**示例**：
```
王经理您好，感谢今天的面试交流！
关于 XX缓存 缓存穿透的问题，我补充一下：我当时项目中用的是布隆过滤器+空值缓存的双重方案，有效避免了穿透问题。
期待后续消息，祝工作顺利！
```

## 五、Offer 谈薪模板

> 完整的薪资谈判策略见 `SKILL-SALARY.md`

```
您好，非常感谢贵司的 Offer！
我对这个岗位非常期待，想和您沟通一下薪资方面：
基于我的[具体技能/项目经验/市场调研]，是否可以在[具体项]上再协商一下？
期待您的回复！
```

## 六、话术规则

- 首次打招呼控制在 **80 字以内**，HR 在消息列表只看前两行
- 不用"您好，我对贵司很感兴趣"开头 — 太泛，HR 每天收到几百条
- 第一句话必须包含**岗位名+1个硬技能**，让 HR 3 秒内判断是否值得点开
- 不发语音、不发表情包、不说"在吗"
- 已读不回最多跟 1 次，不纠缠
- 面试后感谢在 **2 小时内**发送，补充面试中讨论的技术点
- 谈薪时先表达感谢和期待，再提出具体诉求

## 七、与 career-tracker.md 的协作

| 场景 | 操作 |
|------|------|
| 首次投递 | 从本库选择模板 + 用 resume-optimizer 生成核心优势 → 填入模板 |
| 已读不回 | career-tracker 提醒 → 从本库选择跟进模板 |
| 面试后 | career-tracker 提醒 → 从本库选择感谢模板 |
| 收到 Offer | career-tracker 记录 → 从本库选择谈薪模板 + SKILL-SALARY 策略 |
